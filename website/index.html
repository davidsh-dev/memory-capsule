How It Works

The technology behind Memory Capsule

1. Audio Capture

Memory Capsule continuously records audio from your microphone in small chunks. The audio is processed in real-time, ensuring privacy by keeping all data local to your device.

Using the sounddevice library, we capture high-quality audio at 16kHz, optimized for speech recognition.

2. Speech Transcription

The captured audio is transcribed using OpenAI's Whisper, a state-of-the-art speech recognition model that's robust to accents, background noise, and technical language.

We use the "tiny" model by default for speed, but you can choose larger models for increased accuracy.

3. Speaker Diarization

Memory Capsule identifies who said what using advanced speaker diarization technology. This allows the system to attribute each piece of transcribed text to the correct speaker.

We use neural network models to create voice embeddings that can distinguish between different speakers even in overlapping speech.

4. Conversation Storage

All transcribed conversations are stored in a SQLite database with timestamps, speaker information, and other metadata. This creates a searchable archive of all your spoken interactions.

The database is optimized for quick retrieval and can handle years of conversation history.

5. Memory Search & AI Assistant

When you ask a question, Memory Capsule uses both keyword and semantic search to find relevant past conversations. These are fed into a language model using Retrieval-Augmented Generation (RAG).

The AI assistant can then provide contextually aware responses based on your conversation history, speaking back to you using text-to-speech technology.

Installation

Get up and running in minutes

Prerequisites

  • Python 3.8 or higher
  • Anaconda or Miniconda (recommended)
  • Microphone for audio input
  • Speakers for audio output

Installation Steps

# Clone the repository
git clone https://github.com/yourusername/memory-capsule.git
cd memory-capsule

# Create a new Anaconda environment
conda create -n memory-capsule python=3.10
conda activate memory-capsule

# Install the required dependencies
pip install -r requirements.txt

Running Memory Capsule

# Run with default settings
python memory_capsule/run.py

# Or with custom options
python memory_capsule/run.py --whisper-model base --use-local-llm
